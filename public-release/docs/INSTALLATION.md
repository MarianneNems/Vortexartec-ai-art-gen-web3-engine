# Installation Guide

## Prerequisites

- WordPress 5.0 or higher
- PHP 7.4 or higher
- MySQL 5.7 or higher
- 256MB+ PHP memory limit

## Installation Steps

1. **Download the plugin**
2. **Extract to wp-content/plugins/**
3. **Configure environment variables**
4. **Activate in WordPress admin**
5. **Run initial setup wizard**
